# Introducing Odin

Odin is a secure and scalable code execution engine that supports multiple programming languages along with their dependencies and system packages. Whether you are developing a competitive programming platform, an e-learning platform, an interview platform, or an online code editor, Odin can help you achieve your goals by providing efficient code execution capabilities for various workflows.

## What sets Odin apart
